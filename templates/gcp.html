About Google Cloud Platform (GCP)

About GCP

  • GCP is a cloud computing service provided by Google.
  • It can be used to store data, develop and house web applications, and much more.

GCP and this web application

  • This web application was developed using Google App Engine, which is a web development service provided by GCP.
  • All of the files that make this application run on the back end are contained in a bucket in Google Cloud Storage.
  • This was the first time that I developed a web application at all, and I found resources provided by GCP to be very useful.
